# Content Compliance — EULA, IP, Ratings

## 1. EULA Compliance (Mojang/Microsoft)

### Allowed
- Original skin designs
- Fan art (non-commercial gray area)
- Parody/tribute content (must be transformative)
- Generic fantasy/anime/monster themes
- Original characters and worlds

### NOT Allowed (without license)
- **Trademarked characters**: Mickey Mouse, SpongeBob, etc.
- **Copyrighted designs**: Exact replicas from movies/games
- **Brand logos**: Nike, Apple, Gucci, etc.
- **Celebrity names**: Using real names without permission
- **Other game IP**: Exact Fortnite, Valorant, League skins
- **Music lyrics**: Copyrighted song text
- **Offensive content**: Hate speech, violence, drugs

## 2. Age Rating (ESRB)

Marketplace requires E for Everyone content:

- **No blood/gore** in skins or textures
- **No sexual content** (no lingerie, bikinis as skin)
- **No drug/alcohol references**
- **No violence** in pack descriptions
- **No profanity** in names or descriptions
- **No horror that depicts real trauma**
- Skull motifs are OK (Minecraft has skeletons)
- Fantasy violence is OK (swords, armor)

## 3. Description Compliance

Store descriptions must be:

- **Accurate**: Don't claim features that don't exist
- **Original**: No copied text from other creators
- **Simple**: Easy to read for all ages
- **No price mentions**: Don't say "on sale" etc.
- **No external links**: Don't link to Discord etc.
- **No review manipulation**: Don't ask for 5 stars

### Template
```
[Pack Name] brings [N] amazing [type] to your Minecraft world!
[Brief description of what's included]
[Highlight 1-2 best features]
Perfect for [target audience/use case]
```

## 4. Screenshot Guidelines

- Must show **actual in-game content**
- No misleading angles or editing
- Show variety (multiple skins/scenes)
- Clean backgrounds (no clutter)
- Proper lighting
- Family-friendly
- 1280x720 resolution minimum

## 6. Cultural Sensitivity

Marketplace serves 212M players globally:

- Check skin names for unintended meanings
- Avoid religious symbols (crosses, etc.)
- Avoid political symbols (flags can be OK for sports)
- Cultural themes should be respectful (not caricatures)
- Be mindful of different cultural norms
- Our African, Celtic, Aztec packs use respectful design
- Japanese/Korean fashion packs celebrate culture
