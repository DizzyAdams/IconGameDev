# Marketing Assets Guide

## 1. Required per Pack

| Asset | Size | Format | Notes |
|-------|------|--------|-------|
| Pack Icon | 256x256 | PNG | Transparent BG, readable at small size |
| Thumbnail | 300x300 | PNG | Main store thumbnail |
| Screenshot 1-4 | 1280x720 | PNG | Gameplay showcase |
| Hero Banner | 1920x1080 | PNG | Top of store page |

## 2. Thumbnail Design Rules

- Bright colors, high contrast
- Shows the product clearly
- Text limited to 3 words max (pack name)
- No Minecraft logo or trademark misuse
- Consistent branding across all packs
- File under 500KB

## 3. Screenshot Guidelines

- 4 minimum, 6 maximum per pack
- First screenshot is the hero image
- Show variety (different skins, biomes, angles)
- No UI elements covering the content
- 1280x720 minimum, 16:9 aspect ratio
- JPEG or PNG, under 2MB each

## 4. Store Description Tips

- First 100 characters must hook the buyer
- Use bullet points for features
- Include search keywords naturally
- Mention Bedrock Edition compatibility
- Keep between 100-300 characters

## 5. SEO Keywords (PT-BR Market)

- pele de minecraft, skin minecraft, pacote de textura
- mundo minecraft, mapa minecraft, survival, pvp
- bedrock edition, celular, console, pc
- anime, rpg, fantasia, medieval, cyberpunk

## 6. Branding Assets

- "IconMyBedrockMods" logo in store page
- Consistent color scheme: purple (#6366f1), green (#22c55e)
- Social links: YouTube, Discord, TikTok for BR audience
- Watermark screenshots lightly with brand name

## 7. Seasonal Marketing

- Release themed packs 2 weeks before holidays
- Christmas: October-November production
- Valentine's: January production
- Easter: February-March production
- Back to School: July-August production
- Halloween: September-October production
