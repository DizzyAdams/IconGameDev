# Microsoft Partner Program — Full Requirements Checklist

## 1. Eligibility

- [ ] Age: 18+ years old
- [ ] Registered business (CNPJ for Brazil, LLC for US, etc.)
- [ ] Valid tax ID (CPF/CNPJ for Brazil, SSN/EIN for US)
- [ ] Bank account or PayPal capable of international payments
- [ ] Microsoft account (outlook.com or custom domain)

## 2. Portfolio Requirements

- [ ] Minimum 3 polished content pieces
- [ ] All content must be **original** (no IP infringement)
- [ ] Each piece must have:
  - Complete .mcpack file
  - Thumbnail image (800x450 recommended)
  - Pack icon (256x256)
  - Store description (150-300 chars)
  - Screenshots (4-6 per pack, 1280x720)
  - Key art / Hero banner (1920x1080)

## 3. Business Requirements

- [ ] Legal business name matching tax documents
- [ ] Business address
- [ ] Tax identification number
- [ ] Payment method configured
- [ ] Signed partner agreement
- [ ] W-8BEN (international) or W-9 (US) tax form

## 4. Technical Requirements

- [ ] manifest.json v2 format
- [ ] Valid UUIDs (version 4)
- [ ] Proper module types (skin_pack, resources, world_template)
- [ ] min_engine_version >= 1.20.0
- [ ] All texture files in RGBA PNG format
- [ ] Skin dimensions: 64x64 or 128x128
- [ ] Pack icon: 256x256 PNG
- [ ] No external dependencies
- [ ] World templates include valid level.dat

## 5. Content Compliance

- [ ] No trademarks or copyrighted characters without license
- [ ] No violent, hateful, or inappropriate content
- [ ] Age rating appropriate (ESRB E for Everyone)
- [ ] Descriptions are accurate (no false claims)
- [ ] Screenshots match actual content
- [ ] No misleading pricing
- [ ] Family-friendly across all markets

## 6. Submission Process

1. Go to https://creator.microsoft.com
2. Sign in with Microsoft account
3. Click "Apply to Partner Program"
4. Upload portfolio (3-5 best packs)
5. Provide business info
6. Complete tax forms
7. Set up payment method
8. Submit and wait 2-4 weeks

## 7. After Acceptance

- [ ] Upload remaining packs through Partner Center
- [ ] Set pricing in Minecoins
- [ ] Write store descriptions per locale
- [ ] Submit for content review (each pack reviewed individually)
- [ ] Monitor ratings and reviews
- [ ] Update packs as needed
- [ ] Track revenue in Partner Center dashboard

## 8. Ongoing Compliance

- [ ] Content updates must pass review
- [ ] Respond to user reviews
- [ ] Maintain professional conduct
- [ ] Keep business information current
- [ ] File taxes on Marketplace income
- [ ] Adhere to Minecraft Usage Guidelines
